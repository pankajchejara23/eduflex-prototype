EduFlex Dashboard Prototype

This prototype showcases a dashboard developed using student interaction data with learning trajectories (or learning path).
Each learning trajectory consists of a series of task activities designed to build a range of key skills.

The dashboard provides an overview of classroom performance across the entire learning path and offers insights into the current state of each student's knowledge, modeled using Bayesian knowledge modeling. By identifying skills that need further development, the dashboard helps teachers focus their efforts where students need it most, ensuring targeted and effective support.
